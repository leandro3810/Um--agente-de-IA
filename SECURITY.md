# Política de Segurança

## Reportar vulnerabilidades
Abra uma issue privada (ou entre em contato com os mantenedores) com:
- descrição do problema
- passos para reprodução
- impacto potencial

## Boas práticas
- Nunca commitar chaves ou tokens.
- Usar variáveis de ambiente para segredos.
- Validar entradas de usuário.
- Monitorar dependências e corrigir CVEs com prioridade.
