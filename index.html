Um Agente de IA

Projeto modular com arquitetura RAG (Retrieval-Augmented Generation), integração com modelos prontos e base para evolução com API, banco vetorial e observabilidade.

Python 3.11+ RAG Modelos Prontos n8n Webhook Segurança

Objetivo

Construir um agente de IA para apoiar projetos em tempo real com recuperação de contexto e geração de respostas.

Arquitetura

Camadas de conhecimento, recuperação e geração com orquestração no núcleo RAGAgent.

Evolução

Pronto para crescer com API REST, banco vetorial, métricas de avaliação e integração de provedores de modelo.

Principais componentes

  • src/agent.py: ingestão, recuperação e geração de resposta.
  • src/training.py: pipeline básico de preparo de dados.
  • src/evaluation.py: métricas iniciais de avaliação.
  • src/projects.py: gerenciamento de projetos com status e busca.

Como executar

python3 -m src.main python3 -m http.server 8000
Base inicial para um agente de IA seguro, modular e pronto para integração com provedores modernos.
