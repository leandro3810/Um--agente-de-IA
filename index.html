Funcionalidades

Tudo que você precisa para um agente de IA de produção

Do núcleo RAG ao gerenciamento de projetos — cada módulo foi pensado para crescer com sua necessidade.

🔍

Recuperação contextual (RAG)

Recuperação por sobreposição de termos com SimpleRetriever. Fácil de trocar por FAISS, Qdrant ou pgvector.

🤖

Modelos plug-and-play

Interface ReadyModel desacopla o núcleo do provedor. Troque OpenAI, Amazon Bedrock (Claude), n8n ou qualquer API sem alterar o agente.

📋

Gestão de projetos

Cadastre, atualize e busque projetos com status e tarefas. Exporte e importe via CSV local ou Amazon S3. Pergunte ao agente com contexto dos seus projetos.

📊

Treinamento e avaliação

Pipeline de preparo de dados e métricas (exact match, context recall) para acompanhar a qualidade do agente.

🛡️

Segurança nativa

Sanitização de texto, prompt com contexto controlado, sem hardcode de segredos e separação clara entre recuperação e geração.

🚀

Pronto para evoluir

Estrutura preparada para API REST (FastAPI), banco vetorial, observabilidade com OpenTelemetry e CI/CD.

Arquitetura

Camadas bem definidas, fáceis de estender

Quatro camadas ortogonais garantem que você possa trocar qualquer peça sem quebrar o restante.

1

Camada de conhecimento

Documentos em memória via Document. Substituível por qualquer banco vetorial.

2

Camada de recuperação

SimpleRetriever por sobreposição de termos. Interface aberta para dense retrieval.

3

Camada de geração

ReadyModel (Protocol) + adaptadores concretos: EchoReadyModel, N8NWebhookModel, OpenAIModel.

4

Orquestração RAG

RAGAgent.ask() une recuperação + montagem de prompt + chamada ao modelo.

src/
  ├─ agent.py # RAGAgent · SimpleRetriever · modelos
  ├─ main.py # execução via CLI
  ├─ training.py # curadoria e split de dados
  ├─ evaluation.py # exact_match · context_recall
  └─ projects.py # ProjectManager · Task · status
tests/
  ├─ test_agent.py
  ├─ test_projects.py
  └─ test_training_evaluation.py
Exemplos de código

Simples de usar, fácil de adaptar

Veja como criar um agente, integrar modelos e gerenciar projetos em poucos minutos.

from src.agent import Document, EchoReadyModel, RAGAgent, SimpleRetriever

# 1. Documentos de conhecimento
docs = [
    Document("1", "O projeto usa arquitetura RAG para recuperar contexto."),
    Document("2", "A geração é feita por um ReadyModel plugável."),
]

# 2. Monte o agente
agent = RAGAgent(
    model=EchoReadyModel(),
    retriever=SimpleRetriever(docs),
)

# 3. Faça uma pergunta
resposta = agent.ask("Como funciona a recuperação de contexto?")
print(resposta)

Integrações

Troque o modelo sem tocar no agente

O protocolo ReadyModel desacopla completamente a geração do restante da pipeline.

Built-in

EchoReadyModel

Modelo de exemplo que ecoa o prompt recebido. Ideal para testes e desenvolvimento local sem dependências externas.

n8n

N8NWebhookModel

Envia o prompt para um workflow n8n via POST e retorna a resposta. Suporta autenticação Bearer token.

OpenAI

OpenAIModel

Integração direta com a API de Chat Completions da OpenAI. Configurável por modelo (padrão: gpt-4o-mini) e timeout.

AWS Bedrock

BedrockModel

Integração com Amazon Bedrock Runtime para modelos Anthropic Claude. Usa credenciais AWS via variáveis de ambiente, perfil ou IAM Role. Requer pip install boto3.
